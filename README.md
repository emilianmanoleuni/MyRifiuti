## Local Start
### Run Server
Navigate to server folder and:
```
npm install
```
Replace files:
* style.css
in
```
client -> node_modules -> v-calendar -> dist -> style.css
```
* VGrid.css
in
```
client -> node_modules -> vuetify -> lib -> components -> VGrid -> VGrid.css
``` 
```
npm start
```
### Run Client
Navigate to client folder and:
```
npm install
```
```
npm run dev
```
